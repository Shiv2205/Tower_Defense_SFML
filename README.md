## How to Use

1. Create `toolchain.cmake` file in root directory and write the following:
```txt
set(CMAKE_CXX_COMPILER /path/to/C++/compiler)
```

2. **Build:**
```sh
python build.py build full
```


3. **Run:**
- For Linux:
```sh
python build.py run
```

- For Windows:
```sh
build.ps1 -type run
```
